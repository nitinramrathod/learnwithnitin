C Programming Language.

It was created in the 1970s by Dennis Ritchie, and remains very widely used and influential. By design, C's features cleanly reflect the capabilities of the targeted CPUs.

C++ Programming Language

Created by Danish computer scientist Bjarne Stroustrup as an extension of the C programming language, or "C with Classes".

JavaScript Programming Language.
